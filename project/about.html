global events

A Global Event is an optional component of a System Reliability Analysis that groups together Actions requiring a system shutdown so that they will be performed at the same time, thereby maximizing the availability of the system. The information for a Global Event is stored in a System Global Event record. When you create a Global Event, you select Actions that may be reschedule an performed as part of he Global Event. The execution of the selected Action is called a Global Event. To be performed with the Global Event, Actions of the Global Event. The next occurrence of any recurring Actions that have been rescheduled and performed as part of a Global Event will be performed according to the values in the Interval and Interval Units fields in the System Action record.
